Página Web

Una página web, o página electrónica, página digital, o ciberpágina​​ es un documento o información electrónica capaz de contener texto, sonido, vídeo, programas, enlaces, imágenes y muchas otras cosas, adaptada para la llamada World Wide Web y que puede ser accedida mediante un navegador web.

Estilo CSS

CSS, en español «Hojas de estilo en cascada», es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado.

Estructura HTML

HTML, siglas en inglés de HyperText Markup Language, hace referencia al lenguaje de marcado para la elaboración de páginas web, es el estándar que se ha impuesto en la visualización de páginas web y es el que todos los navegadores actuales han adoptado.

Primer Paso

Lo primero para tener una pagina web es descargar un programa para crear nuestro código HTML es mi caso utilice Visual Studio Code que me permitió agregar estilo con el lenguaje CSS.

Segundo Paso

Lo segundo es conseguir un dominio para que puedan encontrar nuestra pagina web, existen dominios de paga y gratuitos. Lo aconsejable es uno de paga para evitar anuncios molestos.

Tercer Paso

Y por ultimo solamente queda de disfrutar la pagina web desde un punto de vista usuario y por supuesto darle un cierto mantenimiento cada cierto tiempo para agregar cosas nuevas.
